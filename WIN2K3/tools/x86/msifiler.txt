Syntax: msifiler.exe -d database.msi [-v] [-h] [-s SOURCEDIR]
	-d: the database to update.
	-v: verbose mode.
	-h: populate MsiFileHash table (and create table if it doesn't exist).
	[-s SOURCEDIR]: specifies an alternative directory to find files.
