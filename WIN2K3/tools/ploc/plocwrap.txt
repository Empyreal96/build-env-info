
PLOCWRAP LogFile BinStructure PlocTree FileScript PlocMode
  LogFile: full path to logfile to create
  BinStructure: full path to structure to ploc
  PlocTree: full path to SourceDepot tree (nt\plocbranch\ploc)
  FileScript: File list script to run (whistler.bat, whistler1.bat, etc)
  PlocMode: Mode of ploc (supported 1252, 1251, mirror, 932)
